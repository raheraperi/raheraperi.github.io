Technical Blog - Introduction to JavaScript

10 April 2018

How does JavaScript compare to HTML and CSS?

HTML, or HyperText Markup Language, is used to create the basic structure and content of a webpage.
CSS, or Cascading Style Sheets, is used for the design of a web page – including where everything is placed and how it looks.
JavaScript is used to define the interactive elements of a webpage that help to engage users. It tells the browser what do do when a user interacts with the page. CSS and HTML are both static, while JavaScript enables a web page to be dynamic.

Control Flow + Loops

Control flow statements allow JavaScript web pages to make decisions by executing code based on a condition. An example of this is a morning routine where there are many conditional variables/decisions that have a flow on effect. We use conditional if statements to execute these functions.

Scenario 1: The early bird catches the worm

Condition: wakes up on-time / doesn’t snooze alarm

Condition: eats breakfast before work

Condition: catches early train to work

Scenario 2: Sleep-ins are nice

Condition: wakes up on-time/ doesn’t snooze alarm
False
Condition: eats breakfast before work
True
Condition: catches early train to work
False

Loops enable an application to repeat the same action until a specific criteria has been met. An example of this is getting ready for work i.e. try on different outfits until I am satisfied that I look work-ready.
We use for, while and do ... while loops statements to execute loop functions.

Arrays + Objects

Arrays store sets/lists of values. These are best used when you’re wanting to perform the same operation on all the entries within the set.

Objects, in contrast, allow us to group values - including other objects - together in order to build more complex data sets. Objects are best used when you’re wanting to access data on a more unique basis.

An example of data stored in simple objects would be a library of information that you want to access individually, by name. We call this name to lookup information a key. For this reason, a key lookup is faster than an array loop and therefore would be useful for large data sets.

Functions

Functions are the building blocks of Javascript, without calling a function it would be rendered completely pointless. Essentially, a function is a procedure or set of statements that perform tasks or math operations.

A function is a block of code that executes tasks in a specific order. The following example shows how a basic function is declared and executed to run sayHello:
// Declaring a function
function sayHello () {
console.log('Hello world!')
}

// using a function
sayHello()
