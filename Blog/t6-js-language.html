Technical Blog - Introduction to JavaScript

10 April 2018

Hello, non-tech friend. I'm going to teach you some of the basics of JavaScript syntax!

Parenthesis

Parenthesis are used for calling functions, to surround conditional statements, or for grouping to enforce Order of Operations.
// Declaring a function
function sayHello () {
console.log('Hello world!')
}

// using a function
sayHello()

Square Brackets

These are mostly used for accessing the properties of an Object (or the elements of an Array). In the following example, I'm acessing the third element in my array using myList[2].
var mylist = [1,2,3,4];
alert(mylist[2]);

Single Quotes vs. Double Quotes

Strings should always be enclosed by quotation marks. Quoting style is up to the programmer, and either style has no special semantics over the other. Some sources say that single quotes are more common.
